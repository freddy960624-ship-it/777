# Smart Posture Monitor 🧘 - 智能姿态检测

一个使用 WebRTC 和 TensorFlow.js 的实时姿态检测应用，可以直接在浏览器中运行。

## 功能特性

✨ **实时姿态检测** - 通过网络摄像头进行实时检测  
📊 **统计分析** - 记录良好姿态和驼背时间  
🎯 **智能校准** - 自适应个人姿态标准  
⚠️ **警告提醒** - 驼背超过 2 秒自动警告  
📈 **数据报告** - 生成姿态准确度报告

## 使用步骤

1. **允许摄像头** - 点击允许按钮授予摄像头权限
2. **坐正身体** - 调整坐姿至舒适的直立位置
3. **点击校准** - 点击「校准姿态」按钮
4. **开始检测** - 应用将开始监测您的姿态
5. **查看报告** - 点击「生成报告」查看统计数据

## 状态说明

- 🟢 **姿态良好** - 保持当前姿态
- 🔴 **驼背中** - 请坐直身体
- ⚠️ **警告** - 驼背超过 2 秒时触发
- ⚪ **未检测到人** - 摄像头没有检测到人脸
- 🟡 **需要校准** - 请先点击校准按钮

## 常见问题

### Q: 摄像头无法访问？
A: 
- 检查浏览器是否获得摄像头权限
- 在浏览器设置中允许摄像头访问
- 确保摄像头硬件正常工作

### Q: 检测不准确？
A:
- 确保光线充足
- 保持脸部在摄像头正对位置
- 重新点击校准按钮

### Q: 离线能用吗？
A:
- 第一次需要在线加载模型
- 之后可以离线使用（需缓存）
- 或下载整个项目到本地使用
